# Portfolio — Entreprod

Bienvenue sur le portfolio d’**Entreprod**. Ce site présente nos services de développement web et mobile et met en avant notre expertise sur des technologies modernes telles que React, Next.js, Flutter, React Native, WordPress et Shopify, ainsi que notre utilisation avancée des outils d’intelligence artificielle.

D’autres technologies viendront prochainement compléter cet éventail, et le site est actuellement en cours d’amélioration afin de proposer une présentation toujours plus claire, structurée et alignée avec les standards du développement moderne.

L’ensemble des réalisations et des projets en cours est consultable sur [Instagram @Entreprod](https://www.instagram.com/Entreprod).

## Installation & développement local

### Installer les dépendances

```bash
npm install
```

### Lancer en mode développement

```bash
npm run dev
```

### Compiler pour la production

```bash
npm run build
```

### Lancer en mode production

```bash
npm start
```

### Linter

```bash
npm run lint
```

## Déploiement sur Vercel

1. Push ce repo sur GitHub
1. Connecte-le sur [vercel.com](https://vercel.com) → Import Git Repository
1. Déploie (URL live en quelques secondes)
